# Category S7 Class

An S7 class representing a category for organizing todos.

## Usage

``` r
Category(
  id = "01KETGSZVZQYSQ2ZVET4E1ZV71",
  title = character(0),
  description = character(0)
)
```

## Arguments

- id:

  ULID unique identifier for the category.

- title:

  Character title of the category.

- description:

  Character description of the category.
